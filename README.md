# 👨‍💻 Sobre

Este portfólio reúne projetos práticos voltados para análise de dados, visualização e automação com IA.
Cada projeto foi desenvolvido com foco em resolver problemas reais ou simular cenários profissionais.

O objetivo é demonstrar:

* Limpeza e tratamento de dados
* Análise exploratória
* Construção de dashboards
* Aplicações com IA
* Publicação de apps com Streamlit
* Organização de projetos
